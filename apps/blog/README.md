# Blog CMS

A modern blog platform with PHP/MySQL backend.

## Features
- Post management with categories
- Comments system
- Admin panel with CRUD
- View counter
- Responsive design

## Quick Start
1. Start XAMPP (Apache + MySQL)
2. Visit: `http://localhost/Portfolio/apps/04-blog-cms/`
3. Database auto-creates with sample posts

## Structure
```
04-blog-cms/
├── index.php      # Blog listing
├── post.php       # Single post + comments
├── admin/         # Post management
├── includes/db.php
└── css/style.css
```
