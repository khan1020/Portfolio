Why I Built This

I was using a spreadsheet to track expenses. It worked, but I wanted automatic charts and easier data entry. This was my solution.

The key insight: seeing a pie chart of where money goes is way more impactful than reading rows of numbers.

Features

Add Transactions

Income or expense. Amount, category, date, optional note.

Category Breakdown

Pie chart showing spending by category. Instantly see where money goes.

Balance Tracking

Running balance updates automatically. Color-coded: green good, red bad.

Date Filtering

View this week, this month, or custom date ranges.

The Chart Data Query

Aggregating expenses by category for the pie chart:

$stmt = $pdo->prepare(" SELECT category, SUM(amount) as total FROM transactions WHERE type = 'expense' AND created_at >= ? GROUP BY category ORDER BY total DESC "); $stmt->execute([$start_date]); $categories = $stmt->fetchAll();
