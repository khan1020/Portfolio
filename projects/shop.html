Why I Built This

Honestly? I was tired of seeing "Add to Cart" buttons that didn't actually work in portfolio demos. When I was learning web development, every tutorial showed the frontend but skipped the hard parts â€” the database, the session handling, the checkout flow.

What It Does

This isn't just a product display page. It's a mini e-commerce platform with:

Product Catalog

Browse products with filtering by category. Pagination for large catalogs. Clean grid layout.

Working Cart

Add items, update quantities, remove products. Cart persists across page refreshes using sessions.

Checkout Flow

Complete checkout with customer details form. Order summary and confirmation page.

Admin Panel

Add, edit, delete products. Manage categories. View orders. Protected with login.

The Development Journey

Day 1-2: Database Design

Day 7: Polish & Testing
