The Problem I Solved

A friend runs a small restaurant. They were taking reservations over WhatsApp. Double bookings happened. Time slots were unclear. Customers got frustrated.

I built them this system. Now customers see available slots, pick one, fill out their details, and get confirmation. The restaurant sees all bookings in one place.

How It Works

Date Selection

Pick a date. System shows only available time slots.

Time Slots

Pre-defined slots (12PM, 1PM, 2PM...). Each slot has limited capacity.

Party Size

Select number of guests. System checks if slot can accommodate.

Confirmation

Booking saved to database. Confirmation number generated.
