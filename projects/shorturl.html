Generating Short Codes

The core challenge: create unique, short, hard-to-guess codes:

function generateShortCode($length = 6) { $chars = '0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ'; do { $code = ''; for ($i = 0; $i < $length; $i++) { $code .=$chars[random_int(0, strlen($chars) - 1)]; } } while (codeExists($code)); // Keep trying until unique return $code; }
