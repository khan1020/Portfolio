Chart.js Integration

The charts pull real data from MySQL, format it as JSON, and Chart.js renders it. Here's the basic flow:

// PHP: Get revenue data for last 7 days $data = []; for ($i = 6; $i >= 0; $i--) { $date = date('Y-m-d', strtotime("-$i days")); $stmt = $pdo->prepare("SELECT SUM(amount) FROM orders WHERE DATE(created_at) = ?"); $stmt->execute([$date]); $data[] = [ 'date' => date('M d', strtotime($date)), 'revenue' => $stmt->fetchColumn() ?? 0 ]; }
