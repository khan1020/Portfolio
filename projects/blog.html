The Slug System

URLs like /post?id=47 are ugly. I wanted /post/my-first-blog-post. Here's how I generate slugs:

function createSlug($title) { // Convert to lowercase, replace spaces with hyphens $slug = strtolower(trim($title)); $slug = preg_replace('/[^a-z0-9-]/', '-', $slug); $slug = preg_replace('/-+/', '-', $slug); // Make unique if slug already exists $original = $slug; $count = 1; while (slugExists($slug)) { $slug = $original . '-' . $count++; } return $slug; }

Lessons I Learned

üìù Content is king, but UX is queen

A beautiful design makes people want to read. Wide content areas, good typography, and proper spacing matter more than fancy animations.

üìù Slugs need to be unique

Two posts can have the same title. The slug (URL) can't be duplicated. Always check and append numbers if needed.

üìù View counts need de-duplication

Without session tracking, refreshing a page adds 10 views. I track views per session to get accurate counts.
