The Follow System Database

A simple junction table that enables powerful queries:

-- Who follows whom CREATE TABLE follows ( follower_id INT, followed_id INT, created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP, PRIMARY KEY (follower_id, followed_id) ); -- Get posts from people I follow SELECT posts.* FROM posts JOIN follows ON posts.user_id = follows.followed_id WHERE follows.follower_id = ? ORDER BY posts.created_at DESC

The Like Toggle

AJAX like button with INSERT/DELETE toggle:

// Check if already liked, then toggle $exists = $pdo->prepare("SELECT 1 FROM likes WHERE user_id = ? AND post_id = ?"); $exists->execute([$user_id, $post_id]); if ($exists->fetch()) { // Unlike $pdo->prepare("DELETE FROM likes WHERE user_id = ? AND post_id = ?")->execute([$user_id, $post_id]); echo json_encode(['liked' => false]); } else { // Like $pdo->prepare("INSERT INTO likes (user_id, post_id) VALUES (?, ?)")->execute([$user_id, $post_id]); echo json_encode(['liked' => true]); }
