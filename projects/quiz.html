Making Learning Fun

I was studying for an exam and wished I had a quick way to test myself. Flashcard apps are great, but sometimes you want multiple choice with a timer to add pressure.

This quiz app scratches that itch. Questions from a database, random order, countdown timer, instant feedback on each answer.

Features

Randomized Questions

Questions pulled randomly from database. Different quiz every time.

Countdown Timer

30 seconds per question. Creates urgency and realism.

Instant Feedback

Green for correct, red for wrong. See the right answer immediately.

Score Summary

Final score with percentage. Leaderboard (coming soon!).
